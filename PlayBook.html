PlayBook

Modular tabletop game-in-a-book designed for portability, storage, and customizable gameplay. Built for the McMaster Design League Tabletop Toys Competition.

← Back

Project Overview

PlayBook is a compact tabletop game system designed inside a book form factor. The product stores all game components internally and can be reconfigured to play Hangman, Guess Who, and other memory-based games. The goal was to create a screen-free, portable toy that reduces clutter while increasing replayability.

The Problem

Traditional tabletop games are bulky, easy to lose pieces from, and limited to one game mode. PlayBook addresses this by integrating storage directly into the product and using a modular internal layout to support multiple games within one compact object.
